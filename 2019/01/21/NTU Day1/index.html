0x00 引言

李光耀: 假如我们掌握政权时,不为人民争取更好的生活而奋斗,只是填饱几个人的腰包,我们一定会灭亡.我们的立场是消除过去制度的罪恶,保留良好的东西.政府是根据人民的愿望,把一切需求,勇敢地摆在个人的利益之上.

0x01 今日主题

新加坡社会概况和社会制度

0x02 新加坡概况

  • 常住人口531万,人均占地面积比较小,但是给人的感觉人口总体比较稀疏

  • 城市化水平100%

  • 种族组成:Chinese + Malayas + Indians + others

    提倡多种族平等的政策:HDB混居

  • 生存的核心是人才的教育培养问题

  • 经济发展从劳动密集型最终转向成为知识密集型

0x03 先进经验

任何一个发达的国家都有可以学习的地方,由于新加坡与中国之间不间断的联系以及近乎相似的治国理念,发达如斯的花园城市更是有许多值得借鉴的经验.

治国理念

  • 建设宜居宜业城市
  • 生态
  • 服务(以人为本 提升城市活力)
  • 整合(工业园 高度集成产业)
  • 精品(提升城市形象)
  • 集约人才(教育为本 精英治国)

城市规划

  • 建设卫星城
  • 特点: 科学性,前瞻性,透明性,权威性,和谐性

住房建设

  • 核心思想: “居者有其屋”

  • 月工资的20%偿还贷款,最高可贷款60年

  • 补贴机制

  • 83% 保障性住房

    15% 私人开发用于投资的商品房

  • 住房保障比较好,强调”家”的意识

绿化建设

  • 规划先行 滚动推进
  • 阳台绿化 屋顶花园
  • 法令保障 令行禁止
  • 市场运作 社会参与
  • 凡是有山的建筑都依山就势,保持山景的完整,凡是有水住宅都拥有大片的亲水平台
  • 建造娱乐中心的同时需要建设附带的公园,并且在娱乐设施建设占地不足40%的情况下,建设附带公园占地超过40%

城市管理与交通管理

  • 抑制私人交通:COE拥车证,购买之后只有10年的期限,过期需要重新购买

  • 15-30-45体系:

    市区点对点运输需要15分钟;

    市区到任何地点需要30分钟;

    国家内任何地点之间运输需要45分钟

  • ERP制度:不停车扣费闸门(国内仿造了ETC)

  • 人车、动物分流;

    动态配时(根据大数据生成车辆通过的时间进行配时,提高运作效率)

社会管理

  • 政府主导
  • 强化组织
  • 统一指导
  • 人民民主
  • 沟通联系
  • HDB混居
  • 学校义工
  • 人口老龄化严重

0x04 新加坡教育体制

  • smart classroom 智慧教室问题

  • 教育投资 新加坡大概20+% 中国大概3-4%

  • 只有大概25%的人可以进入大学学习

  • 社会上的艺术工程师和技术工程师也处于急需的状态

  • 以英语为主的双语教育

  • 大学教育第一年级学生保证大学住宿,2、3年级只有课外活动取得优异成果的学生才有大学宿舍

  • 本科教学对素质的要求:

    • 四大能力:会思考,会表达,会自主学习,会面对未知变局
    • 六大软实力:如何找资源,交朋友,合纵联盟,发现关键问题,让人印象深刻,营销创意

  • 大学教育的目的:

    沟通能力,批判性思考,道义的思考,培养公民责任,接受多元化的生活,生活在国际化社会,培养多方面兴趣,为工作做好准备

0x05 一些思考

  • 沟通交流与上访问题

    新加坡的区域议员每周要举行选民接待会,即需要和区域内的选民进行沟通交流,记录选民意见并进行接见.对于学校有突出贡献的学生颁发证书.及时的沟通交流与问题反映机制避免了上访等现象的出现.

  • 家庭概念与个体相互独立

    工作时间和家庭生活时间泾渭分明,在上班时间专心工作,在下班时间专注和家人在一起的时光,很少或者几乎没有不必要的应酬和聚餐,雇佣公司也不会进行打扰.

  • 精英治国

    强调精英治国,以成绩和其他能力的优劣作为判别的依据.欢迎并接纳所有有能力的人才,共同建设新加坡的未来.

  • 智慧教室问题 拆除讲台


    智慧教室问题,在一个能容纳五十人左右的教室中,大概布置了十台左右大小不同的屏幕.屏幕大多以三个为一组,每组都有不同的标号,方便小组讨论的过程中进行投屏.桌子设计成圆角三角形,方便成员的讨论与研究.

    智慧教室拆除讲台,讲究三分教七分学,自然而然消除教师与学生之间的隔阂.上课过程中可以随意走动.

  • 教育观念的转变: thinking school, learning nation——>think less, learn more

  • NTU的宣传片: Technology for better living 教育改变科技,科技改变生活.

  • 校长筛选

    NTU的校长大多为其他世界名校的校长以及诺贝尔奖的获得者,在学术研究和管理学校方面有着多年的经验.

  • 服务代替管理

    这个是国内外大学最与众不同的地方.国内大学讲究管理,设立学院书院体制,下设辅导员班主任班助等一些职务,旨在管理大大小小的学生事务.

    而国外大学则强调自我管理,进入大学的学生已经被视为成年人,出现的问题学校会提供各式各样的帮助服务,但是绝对不会严加管理学生的日常生活,全部靠自己的自律能力.责任方面,举个例子来说如果说学生自杀与学校无任何关系.
    把学生当成独立个体进行培养,尊重其多样性与个性,而不是将他们看作自己的孩子进行管理,这种观念下培养出来的学生会有较强的能力.

  • 是什么让中国人留下来

    简单的社会,隐私的家庭,自由支配的时间,更少的压力,专心的学习
